# Capability Milestones (No Time-Based Planning)

## M1: Agent Control Literacy
- Je kunt tasks/runs/tool calls modelleren in DB
- Je kunt status transitions valideren
- Je begrijpt budget/risk gating rules

## M2: Production Reliability
- Retries zijn geclassificeerd en begrensd
- Workflow recovery voorkomt duplicate effects
- Test suite dekt failure scenarios

## M3: Governance by Default
- Approval flow voor high-risk acties
- Prompt injection mitigations in place
- Volledige audit trail in run events

## M4: Retrieval Competence
- Hybrid retrieval + reranking conceptueel helder
- Memory tiers toegepast op business use-cases
- Evals tonen reproduceerbare quality improvements

## M5: Economics Mastery
- Cost per task en cost-per-success inzichtelijk
- Expected value gating operationeel
- Model tiering gekoppeld aan risk/value

## M6: Multi-Agent Design
- Rollen en handoff protocol expliciet
- Conflict resolution en escalation paths duidelijk
- Budget allocation per agent/team beheerst

## M7: Enterprise Rollout Readiness
- Security/governance runbooks bestaan en werken
- Human approval en exception handling getest
- Backups en recovery paths bewezen in drills
