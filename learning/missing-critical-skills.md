- Build SLO/SLA thinking into agent workflows

- Translate business goals into measurable agent tasks

- Backward-compatible schema changes
- Versioned contracts between services
- Migration safety patterns

## 5) Experiment Design for Agents
- A/B and shadow runs
- Counterfactual analysis
- Decision quality metrics beyond "looks good"

## 6) Trust and UX for Human Approval
- Fast approval UX for high-risk actions
- Explainability formats for non-technical reviewers

## 7) Procurement and Vendor Strategy
- Multi-provider contract resilience
- Exit strategy from provider lock-in
- Cost governance at procurement level, not only code level
