# Capstone A: Ops/Portal Autonomous Improvement

## Goal
Autonoom verbeteringen voorstellen en veilig uitvoeren op een intern portal, met human approval op risicovolle acties.

## Agent team
- Planner agent
- Data quality agent
- UX regression agent
- Governance critic agent

## Required outputs
- Prioritized improvement backlog
- Estimated value/cost per item
- Safe execution plan with rollback
- Audit-ready change report

## Must-pass checks
- No unsafe direct mutations without approval
- Data quality checks against dirty/incomplete records
- Cost and ROI logged per task
