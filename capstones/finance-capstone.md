# Capstone C: Finance Analysis with Audit Controls

## Goal
Agent system voor anomaly detection, reconciliatie en exception triage met volledig audit spoor.

## Agent team
- Reconciliation agent
- Anomaly detector
- Risk reviewer
- Report composer

## Controls
- Immutable event logs
- Approval for postings/adjustments
- Explainable outputs with evidence links
- Strict schema outputs for downstream finance systems
